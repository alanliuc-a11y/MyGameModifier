# 快速编译APK指南

## 方案1: 使用GitHub Actions（最简单）

### 步骤1: 上传代码到GitHub
```bash
# 1. 注册GitHub账号
# 2. 创建新仓库 (New Repository)
# 3. 上传MyGameModifier文件夹
```

### 步骤2: GitHub自动编译
我已经为你创建了`.github/workflows/build.yml`文件

上传后GitHub会自动编译，大约2-3分钟后可以下载APK

### 步骤3: 下载APK
编译完成后在Actions页面下载

---

## 方案2: 使用Android Studio（推荐）

### 步骤1: 下载安装
- 下载: https://developer.android.com/studio
- 安装Android Studio

### 步骤2: 导入项目
1. 打开Android Studio
2. File → Open
3. 选择 `MyGameModifier` 文件夹
4. 等待Gradle同步完成

### 步骤3: 连接手机
1. 用USB连接手机
2. 手机开启开发者模式 + USB调试
3. Android Studio会自动识别

### 步骤4: 编译运行
1. 点击 ▶ Run 按钮
2. 选择你的手机
3. 等待编译安装

---

## 方案3: 使用AIDE（手机端编译）

### 步骤1: 安装AIDE
- 应用商店搜索 "AIDE" 或 "AIDE IDE"
- 安装AIDE - Android IDE

### 步骤2: 导入项目
1. 打开AIDE
2. 菜单 → Open → 选择 `/sdcard/Projects/MyGameModifier/`
3. 等待项目加载

### 步骤3: 编译
1. 菜单 → Build → Build APK
2. 等待编译完成
3. APK会生成在 `app/build/outputs/apk/debug/`

---

## 方案4: 使用Termux（命令行编译）

### 步骤1: 安装Termux
```
下载: https://f-droid.org/packages/com.termux/
```

### 步骤2: 在Termux执行
```bash
# 安装必要包
pkg update
pkg install openjdk-17 gradle

# 进入项目
cd /sdcard/Projects/MyGameModifier

# 编译
gradle assembleDebug

# APK位置
ls app/build/outputs/apk/debug/
```

---

## 推荐使用方案2 (Android Studio)

原因:
- 功能最全
- 调试方便
- 错误提示清晰
- 免费使用

---

## 编译常见问题

### Q: Gradle同步失败？
- 检查网络连接
- 使用国内镜像源

### Q: 找不到SDK？
- Android Studio会自动下载
- 或手动设置SDK路径

### Q: 编译报错？
- 检查代码语法
- 更新Gradle版本

---

## 编译完成后

APK位置: `app/build/outputs/apk/debug/app-debug.apk`

安装方式:
```bash
adb install app-debug.apk
```

或在手机上直接点击安装

---

**推荐**: 使用Android Studio编译最稳定！
